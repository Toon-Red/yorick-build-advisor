Build Your Own Guide

Want to make your own version for a different champion? Here's the step-by-step.

Define your rune pages

Create named rune page templates in rune_pages.py. Each page has a keystone, primary tree, secondary tree, 6 rune slots, and 3 shard slots. Mark which slots are "adaptable" — they'll change per matchup.

"Grasp-1": RunePageTemplate( name="Grasp-1", primary_style_id=8400, # Resolve sub_style_id=8000, # Precision selected_perk_ids=( GRASP, DEMOLISH, BONE_PLATING, REVITALIZE, LEGEND_ALACRITY, PRESENCE_OF_MIND, SHARD_AS, SHARD_HP, SHARD_TENACITY, ), )

Define your item builds

Create named item build templates in item_builds.py. Each build has starter items, boot options, a core path (ordered), and situational options.

"Default BBC": ItemBuildTemplate( name="Default BBC", starter=(DORANS_BLADE, HEALTH_POTION), boots=(PLATED_STEELCAPS, MERCURY_TREADS, IONIAN_BOOTS), core=(SPEAR_OF_SHOJIN, BLACK_CLEAVER, SUNDERED_SKY), situational=(HULLBREAKER, STERAKS_GAGE, DEATHS_DANCE, SPIRIT_VISAGE), )

Put enemies into buckets

In rules.py, group enemies by archetype. Define what each bucket means for resolve runes, shards, starters, and item paths.

RANGED_POKE_CHAMPS = {"Teemo", "Quinn", "Vayne", "Kennen", ...} BURST_CC_CHAMPS = {"Riven", "Renekton", "Jax", "Volibear", ...} MS_SHARD_CHAMPS = {"Jax", "Tryndamere", "Yone", "Yasuo", ...} EXHAUST_PRIMARY = {"Tryndamere", "Riven", "Renekton"}

Add specific matchups

In matchup_table.py, give each enemy a difficulty rating, priority-ordered keystone list, item category override, shard override, exhaust recommendation, and advice text.

"Jax": MatchupInfo( difficulty="HARD", keystones=("Grasp-1", "Comet", "Aery"), item_category="iceborn_cleaver", shard_override="MS", exhaust_viable=True, advice="Iceborn rush. W him when he Es...", )
